# Methods -- Generated 2026-05-05 by TaxaWorkflow

Taxonomic assignments were reviewed for ecological plausibility using the TaxaID ecosystem (TaxaAssign, TaxaFlag, TaxaTools). The input consisted of least-common-ancestor consensus assignments from 12S MiFish-U eDNA metabarcoding data. For each ASV, the finest resolved taxonomic name was extracted (species when available, otherwise genus, family, etc.). Site context was inferred automatically using build_context() (TaxaAssign), which queried a large language model (Anthropic Claude, via TaxaTools::call_anthropic_api) with the list of detected taxa and the geographic hint 'Ventura County estuaries, southern California, USA' to characterize the expected habitat, ecoregion, and assemblage composition. Each assignment was then evaluated for plausibility using review_assignments() (TaxaFlag), which submitted each taxon–context pair to the LLM for expert-level review. Assignments deemed implausible for the specified estuarine habitat and geographic region were flagged with explanatory notes. This review step does not alter assignments but provides an independent plausibility assessment to guide manual curation.
