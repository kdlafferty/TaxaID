# Methods -- Generated 2026-05-11 by TaxaWizard

Taxonomic assignments were computed using the TaxaID ecosystem (R packages: TaxaTools, TaxaLikely, TaxaExpect, TaxaAssign). Briefly, eDNA match data were loaded from a percent-identity CSV file and unique taxa were extracted using TaxaTools::create_taxon_names(). Reference sequences for all matched families were fetched from NCBI using the MiFish/12S marker via TaxaLikely::fetch_reference_sequences(), and a pairwise distance matrix was constructed using TaxaLikely::build_sequence_matrix(). A likelihood model was trained on this matrix using TaxaLikely::train_likelihood_model() (anchor_perfect = TRUE), and per-hypothesis likelihoods were evaluated for each sample using TaxaLikely::evaluate_likelihoods() (n_sims = 200) followed by TaxaLikely::filter_top_hypotheses(). Spatial prior probabilities were estimated from GBIF occurrence records (2020–2026) within 5 degrees of the sampling site (34.4°N, 119.8°W) using TaxaExpect::build_priors(), with a custom four-category habitat scheme (Estuarine, Subtidal, Rocky Intertidal, Freshwater) and habitat classification assisted by the Claude large language model (Anthropic). The NCBI backbone (ID = 4) was used for taxon name resolution to ensure compatibility with BLAST-derived match data. Likelihoods and priors were joined by site using TaxaAssign::join_priors() and posterior probabilities were computed using TaxaAssign::compute_posterior() (n_sims = 1000). Consensus taxonomic assignments were generated using TaxaAssign::posterior_consensus() with a cumulative posterior threshold of 0.95 and minimum posterior of 0.05, followed by empirical Bayes refinement via TaxaAssign::update_prior_from_consensus(). Final consensus assignments report the lowest resolved taxonomic rank where cumulative posterior probability exceeded 0.95.
