# Methods -- Generated 2026-05-09 by TaxaWizard

Taxonomic consensus assignments were computed from amplicon sequence variant (ASV) match data using TaxaAssign::score_consensus(). Rank-specific percent identity thresholds were applied: species-level assignment required ≥98% match, genus-level ≥95%, and family-level ≥90%. Matches below 85% were excluded from consideration. A maximum score gap of 1 percentage point between the top hit and the next-best hit was enforced; samples with ambiguous top hits within this gap were collapsed to the next higher rank. Taxonomic rank columns were auto-detected using TaxaTools::detect_ranks() and standardized via TaxaTools::create_taxon_names() prior to consensus scoring.
