# Methods -- Generated 2026-05-18 by TaxaWizard

Taxonomic assignments were computed from eDNA match data using a score-based consensus approach implemented in TaxaAssign::score_consensus(). Percent identity scores (column: PercMatch) were used to assign each amplicon sequence variant (ASV) to the lowest possible taxonomic rank meeting defined thresholds (species ≥ 98%, genus ≥ 95%, family ≥ 90%), with a minimum score of 97% and a maximum within-rank score gap of 2% before collapsing to a higher rank. Unique taxa were extracted using TaxaTools::create_taxon_names(). Ecological context for the sampling site (Point Conception, California, coastal marine, 2023) was inferred using TaxaAssign::build_context() via the Anthropic Claude API. Consensus assignments were subsequently reviewed by TaxaFlag::review_assignments(), which applied LLM-based expert evaluation of each assignment in the context of expected fish diversity for the MiFish 12S marker at this site, flagging assignments inconsistent with known species distributions or habitat associations.
