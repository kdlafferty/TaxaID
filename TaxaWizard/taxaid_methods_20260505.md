# Methods -- Generated 2026-05-05 by TaxaWorkflow

Taxonomic assignments were computed using the TaxaID ecosystem in R. Raw eDNA match data (MiFish-U 12S barcode) were read and standardized to a canonical match table format with columns for sample_id, score, and full NCBI-style taxonomy (Kingdom through Species). The finest available taxonomic rank was identified for each row using TaxaTools::create_taxon_names(). Site ecological context (habitat classification, ecoregion) was inferred from detected taxon names and the geographic hint 'Ventura County estuary, southern California, USA' using TaxaAssign::build_context() with the IUCN Level 1 habitat scheme and the Anthropic Claude LLM. Taxonomic assignments were then computed using the LLM-shortcut pipeline (TaxaAssign::assign_taxa_llm()), which derives score-based likelihoods from match scores and queries the LLM for ecological priors, combining them via Bayes' theorem to produce posterior probabilities for each taxonomic hypothesis per ESV. A consensus assignment per ESV was obtained using TaxaAssign::posterior_consensus() with a minimum posterior threshold of 0.80 and a minimum gap of 0.10. Finally, all consensus assignments were reviewed for ecological plausibility using TaxaFlag::review_assignments(), which queries the LLM to evaluate habitat fit, geographic range consistency, contaminant risk, and potential alternative identifications for each assigned taxon.
