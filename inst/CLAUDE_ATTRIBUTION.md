# AI Assistance Attribution

The core algorithms in the TaxaID ecosystem were developed by K.D. Lafferty
over three years of applied eDNA research at USGS. To generalize these
functions for broad use and package them as a reproducible R ecosystem, I used
Claude Code (Anthropic) as a coding assistant. Claude Code assisted with:
refactoring monolithic scripts into modular package functions, writing roxygen
documentation, generating unit and integration tests, creating vignettes,
resolving CRAN compliance issues, and standardizing interfaces across the
seven-package ecosystem. All scientific decisions — model structure, statistical
methodology, hypothesis framework, and ecological interpretation — were made by
the author. Code generated by Claude Code was reviewed and validated by the
author before inclusion.

## Details

- **Tool:** Claude Code CLI (Anthropic), models Claude Opus 4 and Claude Sonnet 4
- **Period of use:** January–April 2026 (Sessions 1–59)
- **Scope:** Code refactoring, documentation, testing, packaging, CRAN compliance
- **Not in scope:** Scientific design, model selection, ecological interpretation
